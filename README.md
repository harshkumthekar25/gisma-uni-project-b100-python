#capital.com Trading simulator 

## Purpose 
A simple stock trading simulator for beginners to practice buying and selling stocks with virtual demo money.

## Features 
- Buy and Sell 5 different stocks
- Track your portfolio value
- View Transaction history
- Save trades to CSV file
- Error handling for invalid inputs

## how to use
1. Enter your name 
2. Choose from menu options 1-6
3. start with 10,000 demo cash
4. try buying low and selling high

## files 
- 'stock.py' - stock class (represents one company)
- 'portfolio.py' - Portfolio class ( holds your stocks)
- 'transaction.py - Transaction class (records trades)
- 'trading_app.py' - Main program
